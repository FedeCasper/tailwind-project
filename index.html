The Simpsons Api

Esta documentación lo ayudará a familiarizarse con los recursos de la API de Los Simpsons y le mostrará cómo realizar diferentes consultas, para que pueda aprovecharla al máximo.

Ir a la API

URL Base

https://apisimpsons.fly.dev/api/personajes

La URL base contiene información sobre todos los recursos de API disponibles. Todas las solicitudes son GET solicitudes y repasan https. Todas las respuestas devolverán datos en formato json. Por cada Personaje podrá ver :

{{name}}:
{{" info del personaje"}}

Más de 600 personajes

En su URL base https://apisimpsons.fly.dev/api/personajes inicialmente presenta 5 personajes por pagina, comenzando desde la pagina uno pero esto puede cambiar con parámetros asignados:

?limit

Personajes que se desean por pagina.

Ej: https://apisimpsons.fly.dev/api/personajes?limit=50

?page

Ppagina inicial donde se comienza.

Ej: https://apisimpsons.fly.dev/api/personajes?page=2
